Install the following and see links for instructions:
1. Intelli J IDEA (Community)
Download link: https://www.jetbrains.com/idea/download/#section=windows
Instructions: https://www.javatpoint.com/installing-intellij-idea-in-kafka
2. Java JDK and JRE
Download link: https://www.oracle.com/ph/java/technologies/javase/javase8-archive-downloads.html
Instructions: https://www.guru99.com/install-java.html
Set all java environment variables: https://www.ibm.com/docs/en/b2b-integrator/5.2?topic=installation-setting-java-variables-in-windows
3. Gradle
Download link and Instructions: https://gradle.org/install/
4. Intelli J IDEA Cucumber support
Instructions: https://jetbrains.com/help/idea/enabling-cucumber-support-in-project.html

Instructions on how to import and run the test suite:
1. Clone the repository in any IDE software like Eclipse or Intellij.
2. Build the project (Right click the project name > Build Module WefoxTechChallenge.
3. To run the suite, right click on the project > Run tests.
